# Changelog

All notable changes to the Claude Code Developer Course are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/). Versioning follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-26

### Added
- 6-module interactive course: Foundations, Security, Skills, Extensions, Agents, Workflows
- Security module: what not to pass to Claude, dangerous commands, .claudeignore, permissions.deny, safe workflow patterns
- 6 roles: Frontend (React/Next.js), Backend (NestJS/Drizzle), QA, DevOps, Marketing (4 sub-specs), Mobile (iOS/Android)
- Marketing sub-specializations: creative, UAM, creative producer, PMM — with role-specific MCP servers, hooks, skills
- Mobile sub-specializations: iOS (Swift/SwiftUI) and Android (Kotlin/Compose) — with platform-specific examples
- Centralized registries: `curriculum/modules.md` and `curriculum/roles.md` as single source of truth
- New Claude Code commands covered: `/effort`, `/fast`, `/teleport`, `/desktop`, `--remote-control`, `/schedule`, `--worktree`
- 3 teaching modes: Sensei, Coach, Copilot
- cclogviewer MCP server for session tracking
- Plugin marketplace for easy installation
- Updated installation guide per official docs (native install, brew, winget — npm deprecated)
